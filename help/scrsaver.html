Screen Saver Settings in Windows 10.

Sown above is the screen that allows configuration of screen savers in my installation of Windows 10. When you see a screen like this, you have been successfull in installing the screen saver. Press the Settings buttion to configure the screen saver. Press Preview to test it out. (Keep you hand well away from the mouse after pressing preview so that the sceen saver will run.)

The screen saver does not use the clock's themes, however the same basic settings that are available for the clock are also available to the screensaver clock. The image below shows the screenh saver configuration dialog. A cursory glanse will show many of the same settings as the clock, however there are some differences.

Screen Saver Configuration Dialog.

In addition to the basic positioning settings, X,Y, cx, cy, the screen saver has 3 positioning options and a speed. The positioning options are: Center, which places the clock at the center of the screen; Fixed, which places the clock at the pre-specified location; Floating, which allows the clock to bounce around the screen. If Floating is selected, then the speed value determihnes how fast the clock moves. 0 is no movement. Speed is essentially the number of pixels the clock will shift at each update.

The screen saver adds two new settings, Trails, and Shadows. Trails means that the previous clock faces are not erased when the clock floats around, thus leaving a trail across the screen. Shadows adds a shadow to the hand shadows to the clock rendering.

The screen saver has three background options: black, color, and desktop. These will set the background image for the screen saver, either black, a color of your choice, or the current desktop. If trails and floating are turned on, then this image will eventually be comvered up by the clock's trail. If trails and floating are turned on, then this image will eventually be comvered up by the clock's trail.

Finally, Tick Count is the number of times per second the screen saver will attempt to render the clock. Higher numbers produce smoother floating, but costs more performance. This value intereacts with the speed setting, as the speed determines the displacement of the rendering, while this value determines the frequence of drawing. The percieved speed is a combination of both.
